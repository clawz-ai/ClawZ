# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ClawZ, please report it responsibly:

1. **Do NOT** open a public GitHub issue for security vulnerabilities
2. Email the details to the maintainers or use [GitHub Security Advisories](https://github.com/clawz-ai/ClawZ/security/advisories/new)
3. Include steps to reproduce, potential impact, and suggested fix if possible

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

This policy covers the ClawZ desktop application and its build/release infrastructure. For vulnerabilities in the OpenClaw framework itself, please report to the [OpenClaw project](https://openclaw.io).
